Datenschutzerklärung – LAPD Verwaltung Bot

Diese Datenschutzerklärung gilt für den Discord-Bot „LAPD Verwaltung“ (der „Bot“), der auf Servern des Betreibers genutzt wird.

Welche Daten werden gespeichert?

  • Discord User ID (zur Zuordnung von Funktionen wie Dienstnummern/Einträgen)
  • Discord Server ID (zur serverbezogenen Speicherung)
  • Bot-interne Variablen/Einträge (z. B. Dienstnummer, Akteneinträge, Verhaftungs-/Anzeige-Threads)

Wofür werden die Daten genutzt?

Die Daten werden ausschließlich genutzt, um die Funktionen des Bots bereitzustellen (z. B. Akten, Dienstnummern, Verhaftungen, Einträge).

Weitergabe an Dritte

Es findet keine Weitergabe von Daten an Dritte statt.

Speicherdauer

Daten werden so lange gespeichert, wie es für die Bot-Funktionen erforderlich ist oder bis sie durch berechtigte Personen gelöscht werden.

Löschung / Auskunft

Wenn du möchtest, dass deine bot-intern gespeicherten Daten gelöscht werden, wende dich an die Server-Administration.

Kontakt

Kontakt erfolgt über die Server-Administration auf den Servern, auf denen der Bot verwendet wird.
